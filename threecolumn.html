Three Column Layout

This is an example of a three column layout with flexbox

CSS example

Below is column one.

// This is the css.
@media (min-width: 769px) {
.main-header,
.main-nav,
.row {
	display: flex;
}
.main-header {
	flex-direction: column;
	align-items: center;
}
.col {
	flex: 1 50%;
}
.row {
	flex-wrap: wrap;
}
.secondary {
	order: -1;
}
}
@media (min-width: 1025px) {
.main-header {
	flex-direction: row;
	justify-content: space-between;
}
.col {
	flex-basis: 0;
}
.primary {
	flex-grow: 1.4;
}
}
